## 🎯 项目简介
基于支付宝开放平台的增强现实互动营销解决方案，将AR技术与商业场景深度融合。

## 🚀 快速体验
1. 访问：https://lililili-jie.github.io/ar-hongbao
2. 点击"开始游戏"体验AR寻宝
3. 收集5个红包解锁全部奖励

- 支付宝AR识别模拟
- LBS城市地图寻宝
- 支付宝卡券发放系统
- 商户合作展示
